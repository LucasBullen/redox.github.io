How to install RedOx

Rust IDE built in Eclipse

  1. Open Eclipse IDE (Download)
  2. Open the Install New Software Wizard
  3. Enter https://lucasbullen.github.io/redOx/site/ in the site field
  4. Select RedOx - Rust in Eclipse IDE and click Next
  5. Wait for the dependencies to load
  6. Press Next again
  7. Accept the license and press Finish
  8. Restart Eclipse
  9. Start developing your Rust projects in Eclipse IDE!
  10. Make requests and PRs for new features and bug fixes
  11. Learn more about RedOx
